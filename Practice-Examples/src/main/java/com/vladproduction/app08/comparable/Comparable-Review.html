The Comparable<T> interface defines a natural ordering for objects. It is used by sorting algorithms like Collections.sort() and Arrays.sort().

ğŸ”‘ Method

int compareTo(T other);
  • Returns 0 if this == other
  • Returns a negative number if this < other
  • Returns a positive number if this > other

ğŸ§© Example: Person Sort by Age

public class Person implements Comparable<Person> {
    private String name;
    private int age;

    public int compareTo(Person other) {
        return Integer.compare(this.age, other.age); // natural order: by age
    }

    public String toString() {
        return name + " (" + age + ")";
    }
}
List<Person> people = Arrays.asList(
    new Person("Alice", 32),
    new Person("Bob", 24),
    new Person("Charlie", 29)
);

Collections.sort(people); // uses compareTo
