EnumMap in Java

What is EnumMap?

EnumMap is a Map implementation designed for use with enum keys. It's fast, memory-efficient, and preserves enum order.

Key Features

  • Only works with enum as keys.
  • Backed internally by arrays â€” high performance.
  • Faster than HashMap for enum keys.
  • Preserves the natural ordering of enums.

Common Use Cases

  • Storing state machines or workflows
  • Counting enum occurrences
  • Switch-case replacement
  • Day/week/task planners

Example Code

EnumMap<Day, String> schedule = new EnumMap<>(Day.class);
schedule.put(Day.MONDAY, "Gym");
schedule.get(Day.MONDAY);
