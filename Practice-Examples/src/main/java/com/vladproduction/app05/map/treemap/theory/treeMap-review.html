Java TreeMap Summary

What is TreeMap?

TreeMap is a Map implementation that stores keys in a sorted order using a balanced Red-Black tree structure. It provides O(log n) time complexity for insertions, deletions, and lookups.

Key Characteristics

PropertyDetails
OrderingSorted by natural order or custom Comparator
Null KeysNot allowed
Null ValuesAllowed
PerformanceO(log n) for basic operations
InterfaceImplements NavigableMap

Common Methods

  • put(K key, V value) - Insert or update
  • get(Object key) - Retrieve value
  • remove(Object key) - Remove entry
  • firstKey(), lastKey() - Get smallest/largest key
  • floorKey(K key), ceilingKey(K key) - Closest keys
  • pollFirstEntry(), pollLastEntry() - Remove and return first/last entry

Iteration

for (Map.Entry<K, V> entry : treeMap.entrySet()) {
  System.out.println(entry.getKey() + " => " + entry.getValue());
}

Use Cases

  • Range queries and nearest neighbor lookups
  • Implementing calendar booking systems
  • Sorting by keys automatically

Classic Problems

  • Find K Closest Numbers to Target
  • My Calendar I (non-overlapping intervals)
  • Top K Frequent Elements
